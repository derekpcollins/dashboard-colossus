# Colossus Dashboard

Private monitoring dashboard for Colossus automation systems.

## Features

- **Cron Jobs** — Monitor scheduled automation (Morning Brief, Evening Brief)
- Extensible tab system for future monitoring features

## Local Development

```bash
cd ~/Developer/dashboard
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## Deployment

Deployed to Vercel at [dashboard-colossus.vercel.app](https://dashboard-colossus.vercel.app)
